更年轻时喜欢苹果本，陆续买了两三台，每台价格都死贵（1万~2万块）。大约七八年前果断叛逃，自己组装了性能爆表的台式机，对Windows系统的态度从鄙视变回欣赏。 现在的我，已经FIRE、实践低成本生活、退休后再创业。 **实用主义和长期主义** 是我选择东西时的主要思路。

不讲那些花里胡哨的双屏办公、思维导图、桌面便签、皮肤……以下都是最最实用的，先来个大纲： 1. windows的底层逻辑（管理磁盘、应用图标、文件夹等技巧） 2. 软件工具推荐（精选6款） 3. 打字速度（双拼输入法） 4. 地表最强笔记应用（Obsidian） 5. 最好用的浏览器（Vivaldi） 6. 免费听所有歌的方法（musicFree）

### 理解windows

先说个事，如果觉得电脑卡顿，可以先把硬盘换了。老电脑都爱用机械硬盘，换成固态硬盘等于鸟枪换炮，速度提升非常明显！

**睡眠和休眠的区别**

用苹果本的人从来不关机，这不是谣言。其实用windows也可以不关机的！睡眠和休眠可能很多人分不清差别，稍微涉及点专业知识。休眠会把当前系统“现场”打包装进硬盘，即使断电也能从硬盘里读取现场，打开休眠前的所有窗口、应用、文档。睡眠则只把现场写入内存，但是内存里的东西可能因为断电而丢失。

休眠完全不耗电，睡眠会耗电。休眠中唤醒需要十几秒，但睡眠只要几秒。个人更喜欢休眠！部分笔记本有个混合睡眠的模式，会根据情况使用不同睡眠模式，也可以试试。

**C、D、E盘** 😵

硬盘分区是windows独有的，这是个历史遗留问题，其实可以不分。大部分时候的C、D、E盘并不是物理意义的区别，而是安装在同一块硬盘上的软件意义上的磁盘划分。如果台式机有一个固态硬盘、一个机械硬盘，一定要把操作系统安装在更快的硬盘上（固态硬盘），有些会频繁读写文件、对磁盘速度要求高的软件和游戏也是如此。

其实不存在所谓“系统盘”的概念，操作系统本身的载体就是一堆文件，只要安装在速度快的硬盘上就可以了。windows系统（macOS也一样的）的特点是用着用着系统文件会变大，系统盘爆满导致无法安装更多软件。所以最好就不要分磁盘。

不分磁盘，那怎么整理文件？用文件夹啊！

**应用图标**

用好这三个层级，能高效摆放应用图标，快速打开任何需要用到的应用：

- **底部栏** ：这是优先级最高的“地块”，因为大部分时候都有窗口打开，覆盖了桌面空间，只有底部栏是常驻的。
- **桌面** ：优先级仅次于底部栏，一开电脑映入眼帘的就是桌面。桌面东西要尽量少，个人而言只会在桌面放几个文件夹，都是正在进行中的文件或项目。
- **开始菜单** ：win10之后的开始菜单有优化，把“最近访问”移除后，现在的设计看着类似macOS的launchpad，非常适合放快捷图标。

每天都会高频使用的那几个应用，固定在底部栏；较常用的放在开始菜单里；桌面不放看着花里胡哨的图标，安装新软件后生成的桌面图标单独在桌面新建一个 shortcuts 文件夹放里。

**高效使用文件夹**

有这几条重要心得：

- 不要把软件安装在 Program Files 或者 Program Files (x86) 里，不好管理！建一个英文文件夹 installation 专门作为软件安装位置，下面分子文件夹 games, apps, languages, temporary。用英文名是为了安装软件时不出错、能正确寻址。具体怎么分子文件夹要根据自己实际情况来。
- 资源管理器（也就是浏览文件的文件窗口）的左侧可以固定最常用的几个文件夹，例如：桌面、下载、文档、图片、我的项目、installation、回收站等。可以使用“紧凑视图”，让左侧边栏内容排列更紧密。
- 文件夹可以自定义图标（右键→属性→自定义→更改图标），给重要项目换图标可以突出这个文件夹的重要性。

快捷键，通用指其他应用也这样，例如浏览器：

- win+E 打开资源管理器
- win+D 显示桌面
- alt+F4 关闭窗口（通用）
- ctrl+T 新建标签页（通用）
- ctrl+W 关闭标签页（通用）

建的文件夹太多、太深，经常找不到以前下载的文件怎么办？没事，后面有个小工具能帮你。

### 6款装机必备软件

分别是以下6款，都是免费的：

- **Snipaste** ：最好用的截图工具，没有之一。在打开截图模式时，会自动拾取窗口或界面矩形区域，还提供强大的截图标注功能。下载后设置两个快捷键：截图（F1）、贴图（F2）。贴图能把刚刚截图的区域“钉”在屏幕上，在需要参考来自多个应用、文档的内容时非常实用，侧面减少了对多屏幕的依赖。
- **ditto** ：可搜索的剪贴板历史记录。所有复制、剪切的历史内容都被存起来了，包括截图，都可以通过它再调取出来。
- **Everything** ：强大的关键词搜索，快速穿越乱七八糟的文件夹找到目标文件。设置一个快捷键：打开窗口（alt+E）。
- **Sumantra PDF** ：打开超大PDF也很快的阅读器，比美观却臃肿缓慢、经常弹广告的 Drawboard PDF好太多。
- **Readest** ：新晋的电子书阅读器王者，支撑epub、mobi等所有主流格式。字体、版式等调节功能比 Calibre 强很多。目前该开源项目很活跃，新功能也在积极迭代中。
- **oCam** ：录屏神奇，压缩算法出色，生成的视频文件很小，对于有录教程需求的人非常推荐。

前三款最好设置成开机自启动。

噢，还有个杀毒软件，只推荐火绒，别的一概卸载。

### 双拼输入法

我用的是微软双拼，其他双拼方案映射的声母韵母不同，逻辑同时一样的。26个字母+分号，每个键背后都对应着一个声母和一个韵母。

例如：

- 打“龙”字时输入 LS → long，S键对应声母ong
- 打”好“字时输入 HK → hao，K键对应声母ao
- 以此类推。微软双拼比较通用，各个操作系统都能找到（包括手机）

应该花2小时就能学会，用个三四天就熟练了，打字嘎嘎快。缺点是如果一下子要切换回全拼（比如偶尔用别人的设备）的时候会有点不习惯。双拼不像是五笔，或者其他很小众的输入法需要很高成本去切换。双拼本质还是拼音，大多数人还是比较习惯拼音输入的。双拼实现了任何字都只敲两个键就能打出来，因为任何字都由声母和韵母构成。

### Obsidian+AI=高效学习！

Obsidian是基于本地markdown文件的笔记软件，它满足了我很朴素、但大部分笔记软件却做不到的两点需求：

- **快** ：打开笔记的速度取决于电脑本身，只要电脑没问题，它会快得飞起~
- **搜索好用** ：很多笔记软件有搜不到东西的毛病，即使关键词正确、笔记里明明有这个关键词也搜不到！

用好Obsidian需要学习一下markdown语法，只要经常用，其实很简单的，大部分也不需要记，Obsidian还是提供富文本编辑菜单的。

![](https://img9.doubanio.com/view/group_topic/l/public/p707783065.webp)

必备的Obsidian插件：

- **Remotely Save** ：因为Obsidian笔记都是存在本地的一个个.md文件，所以如果要实现多设备同步，需要用它实现。我用的是Dropbox，国内好像大家也用坚果云。同步内容同样可以非对称加密，以保证信息安全。
- **Excalidraw** ：它本身是一个独立的画图工具，被集成到Obsidian里了。它的特点就是可以画出简单、粗糙的图。别小看“简单”、“粗糙”，思考和记录时越简单粗糙才越有效率。大部分场景下你是不需要关注那么多视觉细节的。几百、几千人的大会才需要请专人琢磨PPT的视觉细节，推敲每一个像素。
- **Mousewheel Image Zoom** ：插入图片时默认以全宽呈现，这个很烦。用这个插件可以等比缩小图片。
- **Linter** ：添加这几条custom rules 👇可以让从ChatGPT复制来的内容粘贴到Obsidian时格式不乱
![](https://img9.doubanio.com/view/group_topic/l/public/p707783656.webp)

为什么特地提Obsidian+AI，因为这俩放一起实在太好用了。ChatGPT目前来说，仍然是最好用的AI，回复质量比别的都高，平时只用它。如果你喜欢用别的也没关系，方法都是相通的：

- **提问** ：研究一个新课题时先学会提问，这样才能在ChatGPT那里得到高质量回复。
- **整理** ：把有帮助的内容复制下来（不要全部copy），整理成一篇或多篇笔记放进Obsidian文件夹。
- **复习** ：常用的笔记内容自然会经常打开再看，重复看的时候尝试把内容用自己的语言再写一遍。这样的复习效果是最好的。

这个方法可能适用于任何学科或技能的学习。可以速通。

### 好用但小众浏览器Vivaldi

今年Vivaldi已经支持全平台了，任何手机和电脑都可以用上这个浏览器。这个软件是北欧的公司开发的，做了很多年了，目前没有明确的盈利方式，我给他们捐过款。

Vivaldi的底层是chrome内核，所以chrome能安装的浏览器插件它一样能用。那为什么不用chrome呢？有几大原因：

- **信息安全** ：一个账户管理所有设备的书签、浏览记录、密码等，会同步到云端，但是非对称加密，除了你自己本地的密钥，谁也看不到你的个人信息。
- **全平台** ：这个已经说过了。但是确实大多数浏览器是做不到的，这个非常可贵。如果不能全平台使用，同步数据就不完整了，密码也不是到处都能自动填写。
- **标签管理** ：做项目的时候，浏览器里经常同时打开几十个标签，vivaldi有workspace的概念，可以把标签分组，让标签栏不那么拥挤。还提供锁定标签的功能，减少标签管理的负担，不关闭也不会占用性能。
- **提供非常详细的设置** ：pro用户追求的东西，快捷键、UI自定义、鼠标手势，以及更多你想不到的，应有尽有。

浏览器可能是最重要、最高频使用的软件了。无论你现在用什么，非常建议切换到这款。现在书签、密码这些都可以导入，切换成本不高的。

### 穷鬼听歌福利

musicFree这个小工具本身是个做得很烂的非主流播放器，但是这都不重要。它可以装个神奇的插件，并把所有B站视频转成音频播放。搜索功能也还挺好用的，可以按歌手、按专辑名、歌曲名搜。喜欢的可以收藏，并形成自己的播放列表。

有人可能会问，那为什么不直接用B站？不一样呀，只听歌比乱七八糟的封面和视频的体验要更纯净一些。而且musicFree整个软件的设计逻辑也是基于播放器去做的，更符合听歌这个场景的操作习惯。
